مستودع التدريبات البرمجية اليومية

يهدف المستودع لحفظ التمارين البرمجية و الاكواد التي تم التدرب عليها و ممارسة البرمجة بشكل مستمر يومياً.
معلومات اساسية :
1- لغة البرمجة التي يتم التدرب عليها هي بايثون
2- المساحة البرمجية التي يتم التطبيق عليها هي قوقل كولاب
3- التدريب يسعى لتطوير المهارات البرمجية و التفكير المنطقي و حفظ الاكواد , لذلك لا باس لدي بتكرار الاكواد بهدف اتقانها و حفظها مع تنوع البرامج المنفذة بين الحين و الاخرى لاستكمال المهارات
